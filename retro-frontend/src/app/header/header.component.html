ABOUT THIS PROJECT
Author: Wagner Danda da Silva Filho

This application was built with AI assistance.
PRIMARY TECH STACK
  • Frontend: Angular 21 (standalone components) + NgRx Store/Effects + RxJS + Angular CDK
  • Backend: Node.js + Express 5 + Socket.IO (real-time board sync)
  • Database: Supabase (PostgreSQL via @supabase/supabase-js)
  • Utilities: bcryptjs (password hashing), pdfkit (board export), express-rate-limit
  • Build/Tooling: TypeScript, Angular CLI/build, npm
  • Deployment: Render (single service hosting API, sockets, and built frontend)
